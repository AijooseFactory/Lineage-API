# Lineage API

**Lineage API** is the Python REST API backend for [Lineage](https://github.com/AijooseFactory/Lineage), a genealogy intelligence platform.

It directly reads and writes the Lineage genealogy database — a Gramps-compatible family tree database (SQLite by default, PostgreSQL supported) that uses the same data model as Gramps Desktop. The API is the sole owner of this database and serves as the canonical data layer for all Lineage features including evidence review, dataset generation, and AI-assisted research workflows.

Built on a fork of [Gramps Web API](https://github.com/gramps-project/gramps-web-api).
